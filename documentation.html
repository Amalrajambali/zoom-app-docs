Zoom App Documentation

Overview

This Zoom integration allows users to authorize their Zoom account and create or manage Zoom meetings programmatically.

How to Add the App

  1. Open the Zoom App Marketplace
  2. Click Add App
  3. Authorize the requested permissions

How to Use the App

After authorization, the app can create and manage Zoom meetings based on user actions. Only authorized user data is accessed.

Permissions
